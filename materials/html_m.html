HTML PROGRAMMING STUDY



Apa itu HTML?


  • Singkatan dari HyperText Markup Language
  • Bahasa dasar untuk membuat web


Asal-Usul HTML


Pertama kali dikenalkan oleh seorang ilmuwan fisika bernama Tim Berners-Lee yang sedang bekerja di CERN pada tahun 1980. Karena para peneliti di CERN membutuhkan sebuah cara atau sistem agar bisa saling berbagi dokumen hasil penelitian.



Persiapan



Hal yang dibutuhkan dalam membuat HTML


  • Teks editor untuk menulis HTML; Apapun bisa asalkan dapat digunakan untuk membuat dan menulis dokumen HTML seperti Notepad, Notepad ++
  • Web browser untuk membuka HTML; Bebas menggunakan browser apapun seperti Firefox, Chrome
  • Simpan dokumen dalam bentuk HTML agar dapat dijalankan kedalam web browser; Contohnya belajar.html

Kerangka Dasar



Struktur Dasar HTML


<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>


  • <!DOCTYPE html> Deklarasi tipe dan versi html

  • <html> Elemen pembuka yang wajib ada di setiap dokumen html

  • <head> Elemen untuk membuat judul web

  • <title> Elemen bagian dari head yang akan berisi Judul spesifik yang akan ditunjukkan dalam halaman web

  • <body> Elemen untuk bagian isi

  • <h1> Elemen untuk mendefinisikan besar teks, tersedia dari h1-h6

  • <p>Elemen untuk membuat paragraf

Tag HTML



Tag adalah sebuah penanda awalan dan akhiran dari sebuah elemen di HTML. Tag dibuat dengan kurung siku (<...>), lalu di dalamnya berisi nama tag dan kadang juga ditambahkan dengan atribut.


Tag selalu ditulis berpasangan. Ada tag pembuka dan ada tag penutupnya. Namun, ada juga beberapa tag yang tidak memiliki pasangan penutup. Tag penutup ditulis dengan menambahkan garis miring (/) di depan nama tag


<!DOCTYPE html>
<html>
<h1>This is a Heading</h1>
</html>

Elemen HTML



Elemen adalah sebuah komponen yang menyusun dokumen HTML


  • <br> Elemen untuk pindah baris
  • <hr> Akan membuat garis yang memisahkan

Atribut HTML



Atribut adalah kata khusus yang berada di dalam tag pembuka



  • <a> mendefinisikan hyperlink. Atribut href menentukan URL halaman yang dituju link

  • <img> digunakan untuk menampilkan gambar di halaman HTML. Atribut src menentukan akses ke gambar yang akan ditampilkan

  • <lang> Atribut ini digunakan untuk menambahkan warna, font, ukuran, dan lainnya

  • <style> untuk mendeklarasikan bahasa halaman Web

List



Unordered List


dimulai dengan tag <ul>. Setiap item daftar dimulai dengan tag <li>. Item daftar akan ditandai dengan lingkaran hitam kecil secara default


<ul>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>


Ordered List


dimulai dengan tag <ol>. Setiap item daftar dimulai dengan tag <li>. Item daftar akan ditandai dengan angka secara default



Description List


Tag <dl> mendefinisikan daftar deskripsi, tag <dt> mendefinisikan istilah (nama), dan tag <dd> menjelaskan setiap istilah
